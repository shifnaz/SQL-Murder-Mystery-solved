# SQL-Murder-Mystery-solved

An interactive relational database investigation to solve a crime in "SQL City." This project highlights SQL proficiency using joins, subqueries, and text filtering.

## 🛠️ Tools
- **Database:** SQLite (`sql-murder-mystery.db`)
- **IDE:** VS Code
- **Extensions:** SQLite Viewer / vscode-sqlite

*See `INVESTIGATION.md` for the full walkthrough, queries, and final solution.*
